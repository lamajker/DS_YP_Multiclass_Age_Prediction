# DS_YP_Multiclass_Age_Prediction
Модель предсказания возрастной категории пользователей по цифровому поведению

## О проекте

В рамках проекта построена модель многоклассовой классификации, которая по данным о поведении пользователя в интернете предсказывает его возрастную категорию.

Модель использует агрегированные признаки на основе истории посещений сайтов, а также дополнительные характеристики пользователя.

Задача имеет прикладное значение и может использоваться для сегментации аудитории, персонализации контента и повышения эффективности маркетинговых кампаний.

## Цель

Построить модель, которая определяет возрастную категорию пользователя на основе его цифрового поведения.

## Задача машинного обучения

Тип задачи: многоклассовая классификация

Целевая переменная:
- 0 - младше 18 лет
- 1 - 18-25 лет
- 2 - 26-40 лет
- 3 - 41-55 лет
- 4 - 56+ лет

Основная метрика:
- F1 macro

Критерий успеха:
- F1 macro >= 0.75

## Данные

Используются данные из нескольких источников:

- users - целевая переменная
- visits - история посещений сайтов
- ads_activity - активность взаимодействия с рекламой
- surf_depth - глубина просмотра
- primary_device - основное устройство
- cloud_usage - использование облачных сервисов

Основной источник признаков - таблица visits.

## Подход к решению

1. Объединение данных по user_id
2. Агрегация логов посещений до уровня пользователя
3. Генерация признаков:
   - доли посещений по категориям сайтов
   - доли активности по времени суток
4. Предобработка данных:
   - обработка пропусков
   - кодирование категориальных признаков
   - масштабирование числовых признаков
5. Обучение моделей:
   - логистическая регрессия
   - SVM с различными ядрами
6. Выбор лучшей модели
7. Подбор гиперпараметров
8. Подготовка артефактов для внедрения

## Результаты

Лучшая модель: SVC с RBF-ядром (One-vs-Rest)

Метрики на тестовой выборке:
- Accuracy: 0.9074
- Precision (Macro): 0.9002
- Recall (Macro): 0.8914
- F1 (Macro): 0.8956
- F1 (Weighted): 0.9072

Модель превышает заданный порог качества и показывает устойчивые результаты на всех классах.

## Артефакты

В проекте сохранены:

- best_age_model.joblib - обученный пайплайн модели
- feature_builder.py - функция формирования признаков
- final_predictions.csv - предсказания модели

## Использование модели

1. Загрузить модель:
```
   import joblib
   model = joblib.load("best_age_model.joblib")
````

2. Подготовить данные:

```
   from feature_builder import build_features

   data = build_features(
       users,
       visits,
       ads_activity,
       surf_depth,
       primary_device,
       cloud_usage
   )
   ```

3. Получить предсказания:

```
   predictions = model.predict(data.drop(columns=["age_category"]))
```
